# GAS 部署說明

`Code.gs` 是旅居報名系統的 Apps Script 後端原始碼。

## 第二階段部署

1. 在 Google Sheets 開啟「擴充功能 > Apps Script」。
2. 用本資料夾的 `Code.gs` 完整覆蓋 Apps Script 內的程式碼。
3. 儲存後，在試算表重新整理頁面。
4. 從「幸福旅居系統」選單執行一次「初始化系統」，建立 `11_同行人資料`。
5. 從同一選單執行「設定管理密碼」，輸入至少 8 個字元的新密碼。
6. 從同一選單執行「設定收款帳號」，輸入正式收款帳號。
7. 在 Apps Script 選擇「部署 > 管理部署作業」，編輯現有 Web App。
8. 版本選「新版本」，執行身分選「我」，存取權限沿用目前正式環境設定。
9. 完成部署後確認 Web App 網址未改變，再測試前台報名。

護照照片會存入 Apps Script 執行帳號的 Google Drive 資料夾
`幸福旅居_護照上傳`，檔案維持私人存取，不會自動公開分享。
